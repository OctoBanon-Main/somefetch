
## Simple fetch for unix-like systems

![Screenshot of somefetch](https://github.com/UnixAwesomes/somefetch/blob/master/somefetch.png)

## Install

```
git clone https://github.com/UnixAwesomes/somefetch
cd somefetch
cargo build --release
```
